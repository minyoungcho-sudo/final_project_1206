# 환경 변수 설정 가이드

## .env 파일 설정 방법

1. 프로젝트 루트 디렉토리에 `.env` 파일을 생성합니다.

2. 다음과 같은 형식으로 API 키를 입력합니다:

```
VITE_GPT_API_KEY=sk-your-api-key-here
```

**중요:**
- 반드시 `VITE_` 접두사를 붙여야 합니다
- 등호(`=`) 양쪽에 공백이 없어야 합니다
- 따옴표는 필요하지 않습니다
- API 키는 `sk-`로 시작하는 OpenAI API 키여야 합니다

3. 개발 서버를 **재시작**합니다:
   ```bash
   npm run dev
   ```

## 문제 해결

### API 키가 인식되지 않는 경우:

1. **서버 재시작 확인**: `.env` 파일을 수정한 후 반드시 개발 서버를 재시작해야 합니다.

2. **파일 위치 확인**: `.env` 파일이 프로젝트 루트(`package.json`과 같은 디렉토리)에 있는지 확인하세요.

3. **변수명 확인**: `VITE_GPT_API_KEY` 형식이 정확한지 확인하세요.

4. **브라우저 콘솔 확인**: 브라우저 개발자 도구(F12) 콘솔에서 환경 변수 로딩 로그를 확인하세요.

5. **파일 인코딩**: `.env` 파일이 UTF-8 인코딩으로 저장되어 있는지 확인하세요.
